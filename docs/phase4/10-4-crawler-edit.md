# 10-4: クローラー編集機能

## 基本情報

| 項目 | 内容 |
|------|------|
| **ID** | 10-4 |
| **複雑度** | Medium |
| **見積もり** | 3時間 |
| **依存** | 10-3 |
| **ステータス** | ⬜ 未着手 |

## 説明

既存クローラーの設定を編集する機能を実装する。

## タスク

- [ ] クローラー編集画面ルート
- [ ] 既存データの読み込み
- [ ] 編集フォーム（作成フォームの再利用）
- [ ] 変更検知・保存確認

## 実装詳細

### 編集可能項目

- クローラー名
- スプレッドシート設定
- 広告アカウント選択
- レポート設定（期間、ディメンション、メトリクス）
- スケジュール設定
- タグ

### ファイル構成

```
src/app/(dashboard)/crawlers/[id]/edit/page.tsx
```

### 実装方針

- 新規作成フォームのコンポーネントを再利用
- 編集モードフラグで動作を切り替え
- 保存時に PUT /api/crawlers/[id] を呼び出し

## 完了条件

- [ ] 編集画面が表示される
- [ ] 既存設定が正しく読み込まれる
- [ ] 変更を保存できる
- [ ] スケジュールが再登録される（アクティブの場合）
