## 実装詳細

### ダッシュボードAPI

```typescript
// GET /api/dashboard/stats
interface DashboardStats {
  totalCrawlers: number;
  activeCrawlers: number;
  totalExecutions: number;
  successRate: number;
  recentExecutions: JobExecution[];
  platformStats: {
    platform: string;
    count: number;
    successRate: number;
  }[];
}
```
