# カンタンデータクローラー 開発チケット

## 概要

複数広告プラットフォームからデータを自動取得し、Googleスプレッドシートに出力するSaaSツールの開発チケット管理

## フェーズ構成

| Phase | 期間 | 内容 | チケット数 |
|-------|------|------|-----------|
| [Phase 1](./phase1/) | Week 1-4 | 基盤構築（環境・認証・レイアウト） | 13 |
| [Phase 2](./phase2/) | Week 5-12 | コア機能開発（クローラー作成フロー） | 28 |
| [Phase 3](./phase3/) | Week 13-20 | ジョブ実行・機能拡張 | 22 |
| [Phase 4](./phase4/) | Week 21-24 | 本番準備（テスト・デプロイ） | 13 |

**合計: 76チケット**

## 技術スタック

- **Frontend**: Next.js 14+, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes, Prisma
- **Database**: PostgreSQL
- **Queue**: BullMQ + Redis
- **Auth**: NextAuth.js
